# Kendo.Mvc.UI.Fluent.UploadMessagesBuilder
A builder class for UploadMessages



## Methods

### Cancel(System.String)
Sets the Cancel button text

#### Example

    <%= Html.Kendo().Upload()
        .Name("Upload")
        .Messages(msgs => msgs
        .Cancel("cancel")
        )
    %>
        


#### Parameters

##### cancelMessage `System.String`
New cancel button text.
