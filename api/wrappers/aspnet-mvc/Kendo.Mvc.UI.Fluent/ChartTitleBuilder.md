# Kendo.Mvc.UI.Fluent.ChartTitleBuilder
Defines the fluent interface for configuring the ChartTitle.



## Methods

### Text(System.String)
Sets the title text

#### Example

    <% Html.Kendo().Chart()
        .Name("Chart")
        .Title(title => title.Text("Chart"))
        .Render();
    %>
        


#### Parameters

##### text `System.String`
The text title.




### Font(System.String)

    <% Html.Kendo().Chart()

    %>
        


#### Parameters

##### font `System.String`
The title font (CSS format).
