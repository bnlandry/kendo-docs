# Kendo.Mvc.UI.Fluent.ChartLineBuilder
Defines the fluent interface for configuring ChartLine.



## Methods

### Visible(System.Boolean)
Sets the line visibility

#### Example

    <% Html.Kendo().Chart()
        .Name("Chart")
        .CategoryAxis(axis => axis.MajorGridLines(lines => lines.Visible(true)))
        .Render();
    %>
        


#### Parameters

##### visible `System.Boolean`
The line visibility.
